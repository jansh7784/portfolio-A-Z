Cold Email Generator

  • Tech Stack: Python, Flask, Streamlit, ChromaDB, LangChain, BeautifulSoup
  • GitHub URL: Project Repository
  • Live Demo: Live Project

The Cold Email Generator is a web application designed to automate the process of creating personalized cold emails for job applications. It simplifies the tedious task of writing tailored emails by leveraging AI, web scraping, and portfolio integration.

The project features a user-friendly Streamlit frontend and a Flask backend. Users can upload resumes, scrape job descriptions from URLs, and generate professional emails tailored to specific job opportunities. The application integrates ChromaDB for portfolio querying and LangChain for AI-driven email generation.

The project is deployed on Render, ensuring scalability and reliability. It is designed to handle large volumes of data and provides analytics for tracking popular skills and job types.
